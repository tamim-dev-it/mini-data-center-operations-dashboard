# Mini Data Center Operations Dashboard

A beginner-friendly data center operations project built with Python Flask.  
This dashboard simulates basic data center monitoring tasks such as server availability checks, Linux system health monitoring, incident tracking, security alert review, and asset inventory management.

## Features

- Server online/offline status monitoring
- Basic server inventory using JSON data
- Dashboard UI with summary cards
- Flask-based web application
- Clean HTML/CSS layout for portfolio presentation

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JSON
- Git / GitHub
